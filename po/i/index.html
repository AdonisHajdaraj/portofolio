Hotel Management System

This is a full-stack web application built with React for hotel management, providing separate interfaces for administrators and regular users. The system allows hotel staff to manage rooms, reservations, employees, work schedules, and menu items, while guests can browse available rooms, make reservations, view the restaurant menu, and submit feedback. Administrators have full control over all hotel operations through a dedicated dashboard, including adding, editing, and deleting rooms, managing employee work schedules, tracking reservations, monitoring user feedback, and updating the food menu. The application uses React with Bootstrap for the frontend, Node.js/Express for the backend API, and MySQL for the database, offering a modern, responsive, and user-friendly experience for both hotel staff and guests

E-commerce System

The E-commerce System is a full-stack online shopping platform designed to deliver a secure, scalable, and user-friendly digital commerce experience. The application enables users to register, authenticate securely, browse products by categories, perform advanced searches, and manage their shopping cart with real-time updates. The checkout process is fully integrated with Stripe API to ensure secure online payments and transaction handling. On the administrative side, the system includes a comprehensive dashboard that allows full control over products, inventory, orders, and user management. Administrators can create, update, and delete products, monitor stock levels, track order statuses in real time, and manage customer communications. The dashboard provides a structured overview of platform activity, improving operational efficiency and decision-making. The frontend is developed using React.js to ensure a dynamic and responsive user interface, while the backend is built with Node.js and Express.js following RESTful API architecture principles. MongoDB is used as the database solution to handle scalable data storage and efficient querying. The system follows modern full-stack development practices, including role-based access control, secure authentication flows, and clean separation between client and server logic. This project demonstrates strong understanding of full-stack architecture, API integration, payment processing, state management, and scalable web application design.

React.js Node.js Bootstrap MongoDB MySQL Express.js

Seminar Paper

Analysis of Modern Frontend Technologies: React.js, Vue.js, Angular

This paper presents a comparative analysis of modern frontend technologies: React.js, Vue.js, and Angular. The main objective is to identify the key differences among these frameworks and libraries in terms of architecture, component development approaches, state management, and performance behavior across various projects. React.js, developed by Meta, emphasizes a declarative and flexible approach to building user interfaces, offering significant freedom in selecting additional libraries. Vue.js is known for its simplicity, gentle learning curve, and intuitive structure, while Angular, developed by Google, provides a comprehensive framework with integrated features for large and complex projects. Through theoretical analysis and practical examples, the paper aims to help developers and researchers better understand the purpose, advantages, limitations, and optimal use cases of each technology, enabling the selection of the most suitable tool for different web development projects.
